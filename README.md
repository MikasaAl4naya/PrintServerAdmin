# PrintServerAdmin

WinForms-приложение для администраторов принт-серверов.

## Что реализовано

- Добавление принтера в 3 шага:
  - проверка IP на принт-серверах;
  - выбор драйвера и ввод параметров;
  - финальная проверка и создание принтера;
- Удаление принтера по инвентарному номеру с подтверждением;
- Проверка дублей:
  - по IP на шаге 1;
  - по инвентарному номеру на шаге 2;
- Логирование всех действий в файл.

## Конфиги

Файлы находятся рядом с `exe`:

- `servers_list.json` — список принт-серверов (обычные и TP отдельно);
- `cities.json` — список филиалов (`Name`) и их латинский код (`Code`);
- `server_mapping.json` — соответствие `CityCode (+ optional PrinterType) -> PrintServer`.

Пример `servers_list.json`:

```json
{
  "default": ["printsrv01", "printsrv02"],
  "tp": ["tpprintsrv01"]
}
```

## Логирование

По умолчанию лог пишется в `admin_log.txt` рядом с `exe`.

Формат лога: `Timestamp;User;Action` (CSV с `;`), удобен для импорта в Excel.

Можно задать свой путь через `settings.json`:

```json
{
  "LogFilePath": "\\\\server\\share\\printer_logs\\admin_log.txt"
}
```
